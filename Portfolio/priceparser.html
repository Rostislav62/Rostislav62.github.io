How to Use the Bot

Telegram Bot Commands
  • /start ‚Äî Greet the bot.
  • /subscribe ‚Äî Receive hourly price reports.
  • /unsubscribe ‚Äî Stop receiving updates.
  • /report ‚Äî Get current prices for all tracked products.
  • /reportdelta ‚Äî Show only price changes since last report.
  • /notifychange ‚Äî Toggle notification mode: all updates or only changes.
  • /sort ‚Äî Toggle sorting between name and price.
  • /history ‚Äî View 7-day price chart for each product.
  • /help ‚Äî Display help message with all commands.
Example Usage
  1. Send /start to greet the bot.
  2. Send /report to get current prices for all tracked products.
  3. Send /history to view 7-day price chart for each product.
Web Interface
  • Homepage: http://127.0.0.1:8000
  • Filter by category: http://127.0.0.1:8000?category=Pokemon
  • Sort products: http://127.0.0.1:8000?sort=price
  • Sort + filter example: http://127.0.0.1:8000?category=Pokemon&sort=price
  • Product chart (e.g., Bulbasaur): http://127.0.0.1:8000/history/Bulbasaur
  • JSON API:
    • http://127.0.0.1:8000/api/products ‚Äî All products
    • http://127.0.0.1:8000/api/products?category=Pokemon ‚Äî Filter by category
    • http://127.0.0.1:8000/api/products/Bulbasaur ‚Äî Product details
