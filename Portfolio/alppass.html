AlpPass (Frontend)

What is AlpPass?

AlpPass is the client-side UI for the Mountain Pass Application. This demo page showcases the main user flows of the frontend (English only) and includes screenshots and a short demo video.

Key Features (Frontend)
  • Clean UI for viewing and managing mountain passes
  • Create and edit pass entries using forms
  • Works with the backend REST API (JWT-protected)
  • Two visual themes: light and dark (toggle on this page)
Typical User Flows
  1. Open Home and navigate to passes list.
  2. Browse all passes.
  3. Create a new pass.
  4. Edit an existing pass.
Web Interface (routes)
  • Home: /
  • All passes: /passes
  • Create new pass: /passes/new
